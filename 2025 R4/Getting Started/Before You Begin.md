# Before You Begin

To use Profisee Platform(R) you need:

- A model and entity (and sufficient rights to view them)
- FastApp Studio installed on a workstation running either Windows 10 or Windows 11
- The service URL to access Profisee Platform
- One of the following Microsoft SQL Server versions: SQL Server 2017, 2019, or 2022; Azure SQLDB

Refer to the [Profisee Platform Installation Guide](https://support.profisee.com/wikis/profiseeplatform/deployment_architecture) in the Profisee Support wiki for details on installing Profisee.
