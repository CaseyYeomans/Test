# Profisee Data Management Activities Overview

The **Profisee Data Management** activities that are available in the Visual Studio Toolbox provide the workflow designer the tools they need to read, create, update, and delete data within the entities of the data model.

The table below lists the available activities and provides a summary of their functions and purpose. Refer to each activity in the table above for more specific usage and configuration details.

| | |
| --- | --- |
| **Activity** | **Description** |
| [AddRecord](https://support.profisee.com/wikis/profiseeplatform/add_member) | Supports the addition of a new [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) record to its associated entity. |
| [AddRecords](https://support.profisee.com/wikis/profiseeplatform/add_members) | Supports the addition of a collection of [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) records to their associated entity. |
| [CopyRecordAttribute](https://support.profisee.com/wikis/profiseeplatform/copy_member_attribute) | Allows a value to be copied from a source to a target Attribute for a specified [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) object. |
| [CreateRecordDataContext](https://support.profisee.com/wikis/profiseeplatform/create_member_data_context) | Constructs a new [RecordDataContext](https://support.profisee.com/wikis/profiseeplatform/memberdatacontext_type) for a target entity and Code value that can be used to manage the specific record record in that entity. |
| [CreateSetAddRecord](https://support.profisee.com/wikis/profiseeplatform/create_set_add_member) | Constructs a new [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) object from the dictionary of name/value pairs and adds the [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) to its target entity. |
| [DeleteRecord](https://support.profisee.com/wikis/profiseeplatform/delete_member) | Deletes a specified [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) from its associated entity. |
| [DeleteRecords](https://support.profisee.com/wikis/profiseeplatform/delete_members) | Deletes a one or more records from an entity based on the specification of a collection of Code values. |
| [GetRecord](https://support.profisee.com/wikis/profiseeplatform/get_member) | Reads a [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) from its associated entity. |
| [GetRecords](https://support.profisee.com/wikis/profiseeplatform/get_members) | Reads a collection of [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) objects from an entity based on a filter criteria specified by a [BrowseEntityContext](https://support.profisee.com/wikis/profiseeplatform/create_browse_entity_context). |
| [GetRecordValue<T>](https://support.profisee.com/wikis/profiseeplatform/get_member_value) | Gets a specific [Attribute](https://support.profisee.com/wikis/profiseeplatform/attribute_data_type) from a [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) object based on its Name property. |
| [GetRecordValues](https://support.profisee.com/wikis/profiseeplatform/get_member_values) | Gets a dictionary of name/value pairs from a specified [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) based on a set of Attribute names specified. |
| [GetNewRecordTemplate](https://support.profisee.com/wikis/profiseeplatform/get_new_member_template) | Constructs a new, empty, [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) object as a container for use when adding a new record record to an entity. |
| [GetSetUpdateRecord](https://support.profisee.com/wikis/profiseeplatform/get_set_update_member) | Reads the latest copy of the [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) record from the server and updates its values with the values specified as arguments to the activity. |
| [NewRecordDataContext](https://support.profisee.com/wikis/profiseeplatform/new_member_data_context_activity) | Constructs a new [RecordDataContext](https://support.profisee.com/wikis/profiseeplatform/memberdatacontext_type) base on the configuration provided on the activity during workflow configuration. See [Configuring NewRecordDataContext Activities](https://support.profisee.com/wikis/profiseeplatform/configuring_new_member_data_context_activities). |
| [ReverseRecordTransaction](https://support.profisee.com/wikis/profiseeplatform/reverse_member_transaction_activity) | Reverses (i.e. undoes) a transaction based on the specification of the entity, record Code, and TransactionId. |
| [SetRecordValue](https://support.profisee.com/wikis/profiseeplatform/set_member_value) | Sets a named [Attribute](https://support.profisee.com/wikis/profiseeplatform/attribute_data_type) value for a [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) to a specified value. |
| [SetRecordValues](https://support.profisee.com/wikis/profiseeplatform/set_member_values) | Sets a collection of [Attribute](https://support.profisee.com/wikis/profiseeplatform/attribute_data_type) values for a [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) to those in a specified dictionary of name/value pairs. |
| [UpdateRecord](https://support.profisee.com/wikis/profiseeplatform/update_member) | Updates a [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) in its associated entity on the server. |
| [UpdateRecords](https://support.profisee.com/wikis/profiseeplatform/update_members) | Updates a collection of [Record](https://support.profisee.com/wikis/profiseeplatform/member_data_type) records in a specified entity on the server. |
