# Configure the Report Viewer Control

The Report Viewer control allows an administrator to embed a Power BI report into a FastApp Portal page. Data from an existing Power BI report is displayed in a zone within the portal and updates in real time with Power BI. If you have not already [configured Power BI reports for your Profisee instance](https://support.profisee.com/wikis/profiseeplatform/using_the_power_bi_connector), you must do so before adding a report to a Portal page.

To configure a Report View for a Portal page:

1. Select the Report Viewer control for your portal application.
2. Under **Control Properties**, type a name for the report within the **Name** field.
![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i64896c2535670c58.png)
3. Paste the URL for an existing Power BI report into the **URL** field.

To locate the URL for a Power BI report, navigate to the [Power BI Portal Homepage](https://app.powerbi.com/home) and select a report. On the report, select **File > Embed report > Website or portal**. Copy the link that reads "Here's a link you can use to embed this content."![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i139bcdcea1000ebf.png)

3. Click **Save and Close**.

For more information on using Power BI reports, refer to the [Microsoft Power BI documentation](https://learn.microsoft.com/en-us/power-bi/).
