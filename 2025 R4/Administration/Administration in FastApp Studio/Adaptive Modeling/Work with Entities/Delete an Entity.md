# Delete an Entity

Entities can be deleted from the model when they are no longer needed. To delete an entity:

1. On the new workspace view type tab, click **Entity**.
2. Select the entity you want to delete from the **Entity** menu.
3. In the New Workspace Tab window, click **Settings** next to the **Entity** list.
4. Select **Delete Entity** on the menu.
5. Click **Yes** to confirm the deletion.
6. Type your name in the *Data deletion acknowledgment* window, and then click **OK**.

The entity is removed from the model.
