# Configure Settings on the Name Tab

- To return information about all fields in this tab, use the Attribute Assistant to map your Full Name source attribute to an output attribute, and to automatically configure attributes to contain the information returned in result fields.
- To return information on selected fields, manually configure the output attributes by selecting **Create New** on the menu, or by selecting an existing attribute for each field.

## Options

You can configure several name parsing options for Name verification.

1. Click **Options**.
2. Complete any preferences in the Options section:

- **Correct first name spelling**: Attempt to correct the spelling of the first name
- **Primary name hint**: Choose the most likely expected format of the Full Name source data
- **Gender aggression**: How ambitiously to attempt to assign gender to a neutral first name
- **Gender population**: Set the balance of the data more strongly toward males or females
3. Complete any preferences for the Salutation defaults:

- **Prefix**: The default text to appear before names in salutations
- **Suffix**: The default text to appear after names in salutations
- **Slug**: The default text to use when no name is available
4. Click **OK** to apply the settings.

## Editing Result Mappings

You can map verification service result codes on the Name tab. For more information, see [Editing Result Code Mappings](https://support.profisee.com/wikis/profiseeplatform/edit_result_code_mappings).

Name verification is available for the US and Canada region only.
