# Copy

Copy information from the entity grid or external applications to paste into the entity grid or external applications, such as Excel.

## Copy Cell

1. Highlight the cells you want to copy.
2. Right-click inside the selected range and select **Copy Values** on the menu.

**--or--**

Press **CTRL+C**.

The selected data is copied to the clipboard.

## Copy with Headers

This option is available only from the entity grid. **Copy with Headers** adds the selected cells to the clipboard and includes the column headers matching those in the grid. Use this option to keep data with column headers for offline editing or importing.

1. Select the cells you want to copy in the entity grid.
2. Right-click inside the paste range.
3. Click **Copy with Headers** on the menu.

Selected data is copied to the clipboard with headers.

For information on pasting the copied cells, see [Paste](https://support.profisee.com/wikis/profiseeplatform/Paste).
