# Add a Script Action to a Button in a Form Layout

To add script actions to a form button:

1. [Create a new form](https://support.profisee.com/wikis/profiseeplatform/ConfiguringAForm) or begin [editing an existing form](https://support.profisee.com/wikis/profiseeplatform/edit_form).
2. Add a button to the layout in a form.
3. On the right side of the edit panel, under behavior, click inside the **Script Actions** row.
4. Click the ellipsis (...) next to the **(Collection)** field in the Script Actions row.

The actions selection dialog will appear.
5. Select the script actions you want to use for the button.
6. Configure the script actions as necessary.
7. Continue building your form.
