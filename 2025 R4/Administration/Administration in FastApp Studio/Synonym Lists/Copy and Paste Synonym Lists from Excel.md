# Copy and Paste Synonym Lists from Excel

You can build and maintain lists of synonyms outside of Profisee in Excel. Copy and paste support in the Synonym List maintenance grid lets you copy the lists created in Excel into Profisee.

Excel selections must be three columns or fewer.

1. Using Excel, copy the data from the two columns that contain the synonyms and corresponding replacement terms.
2. Select **Administration** in the navigation panel.
3. Select **Synonym Lists**.
4. Do one of the following:

Click ![](../../../../Images/AddMember.png)**New** and then enter a name for the synonym list in the **Name** field.

**--or--**

Select an existing synonym list, and then click ![](../../../../Images/explorer_edit.png)**Edit**.
5. Select the entire first blank row in the grid by clicking the selection icon ![](../../../../Images/asterisk.PNG) in the first column.
6. With the entire row selected, press **CTRL+V** to paste the synonym and replacement terms into the list.
