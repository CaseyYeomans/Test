# Health Checks

You can diagnose issues with your instance using Health Checks. To do so, navigate to the URL for your Profisee instance with the endpoint **/rest/health**. Here, a status of **Healthy** or **Unhealthy** is displayed for the following databases and services.

- **databases**
- Database Messaging:
- messaging\_db\_health\_check
- logging\_db\_health\_check
- **services**
- Authentication
- Connex
- Data
- File Attachments
- Governance
- Monolith
- Monitor
- Workflow
