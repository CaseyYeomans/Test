# Customizing the Swagger Documentation Page Presentation

## Customizing the Swagger Documentation Page Presentation

Optionally, the REST API documentation page can be configured to display custom branding for use in your environment. To do so, open the REST API service folder and replace the contained images, heading text, and colors with your preferred branding.

- **images\favicon.\*** controls the browser icon
- **images\profisee.png** controls the heading images
- **custom.js** controls the heading content
- **styles.css** controls colors and heading image name
