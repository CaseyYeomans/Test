# Use the Command Line Utility

Perform a variety of Profisee tasks from a command prompt or from batch files with the CLU:

- Process strategies for matching and survivorship, address verification, and data archiving and deployment
- Importing and exporting strategies, rules, lists, views, and model metadata

The name of the CLU file is Profisee.MasterDataMaestro.Utilities.exe, and it is used in a Command Prompt window, batch file, or ETL application.

To run the CLU:

1. Open a Command Prompt window.
2. Navigate to the location where you installed the CLU.

The default installation location is C:\Program Files\Profisee\Master Data Maestro Utilities\versionNumber\

3. Type Profisee.MasterDataMaestro.Utilities.exe at the prompt.
4. The switches and options for the CLU display.

Review the available switches in the Command Line Utility references section to see the specific tasks that can be performed using the Command Line Utility.
