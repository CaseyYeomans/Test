# FastApps Portal Landing Page

The Portal is the web portion of Profisee Platform. The default directory is

http://ProfiseeServerName/ProfiseeVirtualDirectory

When you first open Portal, you will be taken to the FastappPortal landing page, which includes the following sections.

- **Data Browser**

The [Data Browser](https://support.profisee.com/wikis/profiseeplatform/using_the_data_browser) allows you to search and quickly access data within the portal.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i76b8e526eca5aa5b.png)
- **FastApp Library**

The FastApp Library page provides access to all FastApps you have access to. ![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i799047825d00c69b.png)

These FastApps allow users to access their work and any custom pages created by their administrator. You can view and interact with any user data you have created a page for.

- **Recent Notifications**

The Recent Notifications hub displays all current outstanding notifications for the logged-in user. Clicking any of these notifications will open that notification in its corresponding FastApp. If you have no outstanding notifications, a link to the full [Notifications](https://support.profisee.com/wikis/profiseeplatform/viewing_notifications) page will display.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ifc9a1dc335c74372.png)

- **Recent Notifications**

The Recent Notifications hub displays any outstanding notifications for the logged-in user. Clicking any of these notifications will open the [notification](https://support.profisee.com/wikis/profiseeplatform/viewing_notifications) in its corresponding FastApp. You can also select *Click to show all notifications* to go to the full notifications page.
- **My Tasks**

The My Tasks hub displays any outstanding tasks for the logged-in user. Clicking any of these tasks will open the [task](https://support.profisee.com/wikis/profiseeplatform/viewing_tasks) in its corresponding FastApp. If you have no outstanding tasks, no tasks are displayed in the hub.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ib5ccc116b735d874.png)
- **Help and News**

Additionally, at the bottom of the landing page, links to the Help and News sections of support.profisee.com are displayed. These links take the user directly to the Help documentation for the version of the platform in use and to a list of current and upcoming events and features respectively.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ia338aeb2b7d3a9fc.png)
