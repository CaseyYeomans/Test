# Search for Records within a Hierarchy

The hierarchy search function allows users to directly search for and edit records within a hierarchy without needing to expand and collapse the tree view. You can click the **Search**icon ![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i6f5f226ddd97a4f.png) in the top-right of the Hierarchy panel to enter search mode. This feature is disabled during the **Edit** mode.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i7f765a7c6e1edef9.png)

Once the Search function is selected, a menu appears with a search bar and an instructional panel. The Search functions identically to the [Quick Filter](https://support.profisee.com/wikis/profiseeplatform/use_the_quick_filter) feature. Enter a keyword to populate the panel with records in the hierarchy that match the keyword (no records will populate if the keyword returns no results). From this view, you can select a branch record to open it in the standard hierarchy tree and expand the leaves beneath it, or select a leaf record to open it in the **edit** view in the hierarchy tree.

You can also use this function during [Move Mode](https://support.profisee.com/wikis/profiseeplatform/move_record_to_a_new_parent_24r2) to search for branch records when moving leaf nodes.
