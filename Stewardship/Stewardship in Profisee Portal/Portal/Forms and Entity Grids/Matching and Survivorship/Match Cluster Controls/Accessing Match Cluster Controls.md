# Accessing Match Cluster Controls

Match Cluster controls allows a user to access the Survivorship, Split/Combine, and Match Group Visualizer from a single view within a Match Cluster.

To access the Match Cluster controls, click the **Match Cluster** hyperlink on an entity for which you want to view the Match Cluster. This hyperlink can be found on an entity grid, or within the **Edit** view on an entity.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i9aae821d17e7e0bf.png)

Note that the **Enable hyperlink to raise match control** checkbox must be enabled in the Portal page configuration to access the Survivorship view in Profisee Portal.
