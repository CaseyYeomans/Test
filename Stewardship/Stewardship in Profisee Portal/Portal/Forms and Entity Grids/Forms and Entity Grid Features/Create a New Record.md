# Create a New Record

Profisee Administrators create entity grids with associated records using Profisee FastApp Studio. However, as a data steward, you can also manually add new records to an entity grid using the **Add** button.

To manually create a new record:

1. Click the **Add** button on the top toolbar.
![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i7e92effeb39b10c7.png)
2. Enter the Preliminary Information and click **Next.**
![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i8d07e29d3f52fd93.png)
3. If desired, edit the populated attributes and click **Save** or **Save and Close** when you are finished.

The fields available in the form are set by a Profisee administrator. If the entity grid is configured for [Lookup Before Create (LuBC)](https://support.profisee.com/wikis/profiseeplatform/lookup_before_create), you will be notified of any similar existing entities and have the option to Proceed or Cancel the addition.
