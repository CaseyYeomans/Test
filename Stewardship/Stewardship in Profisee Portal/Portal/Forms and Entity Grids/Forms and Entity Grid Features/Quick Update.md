# Quick Update

A user with the **Quick Update User** role or higher can use the Quick Update functions to update attribute values of records within an entity grid. These changes can be made to an individual record, multiple selected records, or all records within an entity grid.

## Quick Update (Selected)

You can use the Quick Update (Selected) function to change the attribute values of selected records within an entity grid.

1. Click the ellipsis icon to the left of the record you want to update to open the dropdown dialogue box. Alternatively, right click the row of the record you want to update to open the dropdown dialogue box.

If you want to update multiple records, CTRL + Click multiple records to highlight them, then click the ellipsis next to one of the highlighted records. Alternatively, right click one of the highlighted records.

2. Click **Quick Update (Selected)** to open the Quick Updated (Selected) menu.
3. Click the dropdown menu under **Attribute** and select an attribute from the list.
4. Enter the desired value for the attribute under the **Value** field.
5. Click **Add Attribute** and repeat steps 3-4 for all desired attributes.
6. Click **Save** to apply updates.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ieaf42772b14738fc.png)

## Quick Update (All)

You can use the Quick Update (All) function to change the attribute values of all records within an entity grid.

1. Click the **Quick Update (All)** icon on the top-right of the entity grid to open the Quick Update (All) menu.
2. Click the dropdown menu under **Attribute** and select an attribute from the list.
3. Enter the desired value for this attribute.
4. Click **Add Attribute** and repeat steps 2-3 for all desired attributes
5. Click **Save** to apply updates.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ibcd69765e6b1b932.png)
