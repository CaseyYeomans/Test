# Add a Child Record to a Hierarchy

Child records can be easily added to a parent from any FastApps Portal page displaying a hierarchy. In a fixed level, parent/child hierarchy, you can add nodes at each level you have permissions to access. In a recursive hierarchy, you can add new levels to an existing hierarchy or add new records to an existing level, building your hierarchy intuitively according to the complexity of your data.

To add a child record to a hierarchy:

1. Navigate to the desired parent level. The level name highlights in green, with bold text.
2. Right-click and select **Add Child**. The Add Record form opens.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i1b339e3a9ff9dbf9.png)
3. Fill in the free-text fields to complete the new record record.
4. Click **Save** to keep the form open and continue editing.
5. Click **Save & Close** to close the form and display the new child record in its hierarchy location.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-iaa284a088e8e6fda.png)
