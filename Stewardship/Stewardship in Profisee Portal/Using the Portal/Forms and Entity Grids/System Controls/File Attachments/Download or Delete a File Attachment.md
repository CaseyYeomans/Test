# Download or Delete a File Attachment

A user with appropriate permissions can download or delete a file attachment from a record within the web portal.

1. Navigate to the field within the web portal that contains the record you want to edit.
2. Right-click the record you want to access and select **Open.** ![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ib3e01c9d3568931f.png)
3. Click the rightmost **View record files** tab.
4. Click the ellipsis to the left of an attached file, then select **Download** or **Delete.**If you select **Delete**, you will be prompted to confirm. Click **Yes** to permanently delete the item, or click **No** to close the prompt.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i1d9f290d3b11ce75.png)
