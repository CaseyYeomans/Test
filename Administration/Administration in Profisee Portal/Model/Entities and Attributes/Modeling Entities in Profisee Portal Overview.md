# Modeling Entities in Profisee Portal Overview

Profisee Portal users with Administrator privileges can perform Advanced Modeling functions in Profisee Portal, without needing to use the Profisee FastApp Studio interface. Using the **Entities** tab in the **Model** section of the portal, you can create, edit, import and export, and manage entities and attributes to be used elsewhere in the Profisee Portal.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i39f8a1462524c14.png)

From here, a list of all created entities is displayed with the following information:

- **Code Generation**: Displays whether the code is automatically generated, or if it is manually entered
- **Created by**: The user who created this entity
- **Created on**: The datetime the entity was created
- **Updated by**: The last user who edited the entity
- **Updated on**: The datetime the entity was last edited

If desired, you can right-click an entity (or select multiple for Delete or Export) in this list to bring up the context menu, allowing you to **Open**, **Clone**, **Delete**, or **Export** the selected entities.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-id6a37650d5ea640.png)

You can also use the Quick Filter to search for specific items in the entity list.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i908923914ee559e4.png)
