# Create, Edit, and Delete Data Products

Administrators in the Profisee Portal can create new data products from the Data Products field in the Administration area of the portal. In any area where Data Products are enabled (such as Strategies), click into the *All Data Products*field, then click *Add new* to open the Create Data Product dialogue and begin.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i52fca3a8ba4135cc.png)

1. Enter a name for your data product in the **Data Product Name** field.
2. Select and move any desired entities in the **Available Entities** field into the **Associated Entities** field to enable them for this data product. If desired, use the quick filter function to quickly search for desired entities.
3. Click **Save** and **Close** to finalize the Data Product.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i37e09850c748ed4b.png)

## Edit a Data Product

You can also edit an existing data product to change which entities are associated with the data product. Navigate to the *All Data Products* field in the Administration area of the portal and click the pencil icon on an existing data product to begin.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i98a78c95a60ae108.png)

From here, you can change the Data Product Name, remove associated entities from the data product, or add new available entities into the data product. When you are finished making edits, click **Save** and **Close** to finalize the changes.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ie3218f9de1688d59.png)

## Delete a Data Product

Existing data products can be deleted by navigating to the *All Data Products* fields and clicking the **X** icon on an existing Data Product. Doing so opens a dialogue to confirm the deletion. Note that deleting a Data Product does **not** delete any of the entities contained within it.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ia6bbb06d3cc2113e.png)
