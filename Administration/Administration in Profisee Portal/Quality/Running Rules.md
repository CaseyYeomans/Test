# Running Rules

Once rules are created, a Profisee Admin can run created rules from the **Quality** section of the Administration view in Profisee Portal. To run rules:

1. Click the **Run** button in the Data Quality Rules toolbar. ![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i490827e9d60281ed.png)
The **Run Rules** pane opens.
![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ia6314899537b06b5.png)
Within this view, rules with the **Running Man** icon are currently running.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ida1b3f826662caa7.png)
2. Select a rule, shift+click to select multiple rules, or click the Select All or Deselect All buttons to select the rules you wish to run.
3. Click **OK** to run all selected rules.
4. Click the **X** icon in the top right of the pane to return to the previous screen.
