# Create an Entity

New entities are created with the Name, Code, Last updated on, Last updated by, Created by, and Created on attributes with no members. New attributes can then be created to populate the entity.

To create an entity:

1. On the new workspace view type tab, click **Entity**.
2. Click **Settings** next to the **Entity** list.
3. Click **Create Entity** on the menu. The Create Entity window opens.
4. Type the name of the new entity in the **Name** field.
5. Optionally, select **Create Code values automatically**, and then enter the value to begin numbering member codes from in the **Starts with** field.
