# Adaptive Modeling Overview

Edit model metadata in real time with Adaptive Modeling. Model changes are available to users immediately. Adaptive Modeling contrasts with Advanced Modeling, which can hold numerous model edits and includes a review option before any changes are applied to a model available on a server. For more information, see [Advanced Modeling overview](https://support.profisee.com/wikis/profiseeplatform/advanced_modeling_overview).
