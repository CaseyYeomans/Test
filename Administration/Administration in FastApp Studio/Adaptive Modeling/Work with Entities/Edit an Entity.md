# Edit an Entity

Entities can be edited to update values. To edit an entity:

1. On the new workspace view type tab, click **Entity**.
2. Select the entity you want to edit from the **Entity** list.
3. In the New Workspace Tab window, click **Settings** next to the **Entity** list.
4. Click **Edit Entity** on the menu.
5. Edit the entity as necessary:

- Change the entity name.
- Select (or clear) the **Create Code values automatically** option.
6. Click **Save**.
