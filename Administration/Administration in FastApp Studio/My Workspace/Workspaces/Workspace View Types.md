# Workspace View Types

You can select from various view types in order to display Profisee data. When a view type is selected, the specified data is retrieved and displayed in a tab in your workspace. Using the **New** **Tab** control, you can open multiple tabs in the workspace of the following view types:

- Entity
- Matching Results

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i88a3f61c21e1aa67.png)

When you select a view type, you are prompted to select necessary information for the view based on the type. For example, if you select an Entity view type, you will be prompted to select the Entity and Member Type Version in order to display the entity grid.
