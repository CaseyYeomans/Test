# Display Columns in the Entity Grid

Right-click the column headers to access the following options for displaying columns in the entity grid.

- **Remove Column**: Removes the selected attribute from the workspace attribute group.
- **Select Columns**: Opens the Select Columns dialog, where you can select entity and parent attributes to appear in the entity grid. The attributes in the Order of selected attributes list are those that display in the grid.

When new columns are added, they appear to the right of the Audit columns.
