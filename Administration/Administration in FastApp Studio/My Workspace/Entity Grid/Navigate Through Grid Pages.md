# Navigate Through Grid Pages

In the bottom right corner of the window, locate the page navigation.

## Jump to Page

In the page navigation, type the page number you want to view and then press **Enter**.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i48c28bb97c7fe859.jpg)

## Skip to Next or Previous Page

Click the single page navigation arrows to move forward or backward one page.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i58f1749b06c3bcfe.jpg)

## Skip to First or Last Page

Click the first or last page navigation arrows to move to the first or last page.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i51a7eeddbf592b25.jpg)
