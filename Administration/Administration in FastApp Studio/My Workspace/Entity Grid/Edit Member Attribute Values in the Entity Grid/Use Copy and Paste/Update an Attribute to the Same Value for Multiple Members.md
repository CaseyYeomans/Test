# Update an Attribute to the Same Value for Multiple Members

To update multiple members with the same attribute value:

1. From an Entity view tab, place the attribute value you want to copy to the clipboard by selecting the value and clicking **Copy** in the entity grid toolbar, or by right-clicking in the cell and selecting **Copy Values** on the menu.
2. Select the target cells to receive the copied value. You can select consecutive attribute cells by clicking the same attribute cell in another member, or by dragging down the column.
3. With the target cells selected, click **Paste** in the toolbar, or right-click and click **Paste Values** on the menu.

The attribute updates with the same value for the selected cells.

**--or--**
4. Use **shift+left mouse** to highlight multiple cells.
5. Type or pick the new value.
6. Press **Enter**.
