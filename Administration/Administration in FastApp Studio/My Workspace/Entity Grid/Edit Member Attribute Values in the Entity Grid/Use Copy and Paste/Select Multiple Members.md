# Select Multiple Members

## Using the Mouse

- **Multiple consecutive members**--Click the row header for the first member and drag to select additional members.
- **Multiple nonconsecutive members**--Press and hold CTRL and click the row headers for multiple members.
- **All members in the grid**--In the top left corner of the grid, click the row header column header.
- **None of the members in the grid**--To unselect the currently selected members, click anywhere inside the entity grid.

## Using Keyboard Shortcuts

- **Expand selection to the adjacent cell**--**SHIFT** + [Arrow]
- **Expand selection to the end of the grid**--**SHIFT** + **CTRL**+ [Arrow]
- **Expand selection to the upper left corner**--**SHIFT** + **CTRL**+ **HOME**
- **Expand selection to the lower right corner**--**SHIFT**+ **CTRL** + **END**

Selecting a member lets you perform actions on that member using the entity grid toolbar and the right-click menu. Possible actions include deleting, advanced editing, cloning, and executing operations.
