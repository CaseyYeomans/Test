# Prerequisites for Workflow Machines

Before beginning the Workflow creation process, ensure your workflow machine has all prerequisite software needed to create a workflow. This software checklist includes:

- Visual Studio version 2015, 2017, 2019, or 2022
- Profisee SDK version that matches your Profisee Platform version
- Profisee Workflow Toolbox (included with the SDK)
- .NET 4.7.2 Framework
- Windows Workflow Foundation
