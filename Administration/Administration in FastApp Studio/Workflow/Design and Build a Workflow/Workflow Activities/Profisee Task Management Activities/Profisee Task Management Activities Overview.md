# Profisee Task Management Activities Overview

The **Profisee Task Management** activities that are available in the Visual Studio Toolbox provide the workflow designer the tools they need to assign work and to notify task participants of those work assignments.

The table below lists the available activities and provides a summary of their functions and purpose. Refer to each activity in the table above for more specific usage and configuration details.

| | |
| --- | --- |
| **Activity** | **Description** |
| [ApprovalTask](http://support.profisee.com/wikis/profiseeplatform/approval_task_activity) | Creates a task and one or more assignments that requests approval (or rejection) of changes to a specific member record. |
| [ContributionTask](http://support.profisee.com/wikis/profiseeplatform/contribution_task_activity) | Creates a task and one or more assignments that request a review and contribution of data for a specified member record. |
| [SendNotification](https://support.profisee.com/wikis/profiseeplatform/send_notification) | Sends an email notification to one or more users based on an event of interest during the workflow's lifetime. |
