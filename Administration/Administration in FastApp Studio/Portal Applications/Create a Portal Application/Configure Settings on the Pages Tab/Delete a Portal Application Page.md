# Delete a Portal Application Page

To delete a portal application page:

1. In the navigation panel, under Administration, click **FastApps**.
2. Access the Pages tab, either by [editing an existing application](https://support.profisee.com/wikis/profiseeplatform/edit_a_portal_application) or [adding a new one](https://support.profisee.com/wikis/profiseeplatform/create_a_new_fastapp).
3. Select the page in the list that you want to delete.
4. Click **Delete** in the Pages tab toolbar.
5. Confirm the deletion.

The page is removed from the list.

If you delete a portal application page, any links to the page are also removed from the portal application menu.
