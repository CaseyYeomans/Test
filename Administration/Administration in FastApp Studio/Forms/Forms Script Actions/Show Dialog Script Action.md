# Show Dialog Script Action

The Show Dialog script action displays a dialog box when the user clicks the button.

To add the Show Dialog script action to a button:

1. From the list of actions, select Show Dialog, and then click Add.

The properties display in the Action Properties panel.
2. In the Dialog Title field, enter the text to display at the top of the dialog.
3. In the Dialog Text field, enter the text to display in the body of the dialog.
4. Click OK.
