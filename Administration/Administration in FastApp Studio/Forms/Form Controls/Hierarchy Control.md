# Hierarchy Control

A hierarchy control allows you to display a member in its specific hierarchical context. This is useful when working with members in an entity grid to quickly see the member's position in the hierarchy.

To configure the hierarchy control, select it in the Layout panel and edit the options in the Display panel.

| Property | Property Option | Description |
| --- | --- | --- |
| **Context** | Hierarchy | Defines the hierarchy to associate with the hierarchy viewer control. |
| **Display** | Display Format (Name | Code | CodeName | NameCode) | |
| | Label (text) | |
| | Label Font Color (RGB value) | |
| | Bold (True / False) | |
| | Italic (True / False) | |
| | Size (Pixel Size) | |
| | Underline (True / False) | |
| **Layout** | Maximum Width (pixels) | |
| | Minimum Width (pixels) | |
