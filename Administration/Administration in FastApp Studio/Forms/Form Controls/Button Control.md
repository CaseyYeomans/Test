# Button Control

The Button control lets you add buttons to your form in addition to the context buttons.

| **Property** | **Property Option** | **Description** |
| --- | --- | --- |
| **Behavior** | Enabled (True | False) | True: The button displays in the form and is enabled. False: The button displays in the form, but it is grayed out and is disabled. |
| | Script Actions | For more information on script actions, see [Use form](https://support.profisee.com/wikis/profiseeplatform/using_form_script_actions) [script actions.](https://support.profisee.com/wikis/profiseeplatform/using_form_script_actions) |
| **Display** | Label (Text) | Defines the label text. |
| | Label Font Color (RGB value) | |
| | Bold (True / False) | |
| | Italic (True / False) | |
| | Size (Pixel Size) | |
| | Underline (True / False) | |
| **Layout** | Alignment (Left | Right) | Aligns the label left or right. |
