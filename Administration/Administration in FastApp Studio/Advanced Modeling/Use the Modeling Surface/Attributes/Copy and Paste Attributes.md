# Copy and Paste Attributes

Copy and paste an attribute using the tree view in Advanced Modeling:

1. Expand the model to the **Entities** container.
2. Expand the entity containing the attribute you want to copy.
3. Expand **Attributes**.
4. Right-click the attribute you want to copy.
5. Click **Copy** on the menu.

The attribute is placed on the clipboard.
6. Expand the destination entity.
7. Right-click the Attributes container in the entity.
8. Click **Paste** on the menu.

The copied attribute appears in the list of attributes.

To copy an attribute using the modeling grid:

1. Expand the model to the **Entities** container.
2. Expand the entity containing the attribute you want to copy.
3. Select the **Attributes** container in the modeling tree.
4. In the modeling grid, select the attribute you want to copy.
5. In the modeling grid, right-click the attribute you want to copy.
6. Click **Copy** on the menu.

The attribute is placed on the clipboard.
7. Expand the destination entity to the **Attributes** container.
8. Select the **Attributes** container in the modeling tree.
9. Right-click inside the modeling grid.
10. Click **Paste** on the menu.

The copied attribute appears in the list of attributes.

The system identity property is not copied along with an attribute when the attribute is copied.
