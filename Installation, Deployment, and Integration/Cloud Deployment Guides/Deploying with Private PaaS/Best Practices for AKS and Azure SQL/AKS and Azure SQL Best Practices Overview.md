# AKS and Azure SQL Best Practices Overview

The following guidelines are based upon the collective learning experience of Profisee Support and Professional Services across many MDM projects and are focused on the features and functions present in Profisee 2023R1 and later. Alternate versions of Profisee may require different approaches. For the best experience, we recommend including a Profisee Consultant in your MDM project.
