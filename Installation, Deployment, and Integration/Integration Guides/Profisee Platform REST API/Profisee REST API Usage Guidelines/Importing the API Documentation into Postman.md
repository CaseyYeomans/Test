# Importing the API Documentation into Postman

If needed, you can import the Swagger OpenAPI JSON directly into Postman using the Swagger link located at the top of the Profisee API documentation page. This allows you to replicate the information stored in your Profisee API documentation page in a local instance of Postman.

1. Open Postman and open the **File** dropdown menu.
2. Select **Import**.
3. Click the **Link** tab.
4. Copy the link address for the Swagger URL located at the top of the Profisee API documentation page. ![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-iba9f0bad43472876.png)
5. Paste the URL into the **Enter a URL** field in Postman.
6. Click **Continue**.
7. Click **Import**.

On import, Postman creates a new Collection containing all calls, values, and descriptions from the Profisee API documentation.
