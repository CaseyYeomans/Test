# Command Line Utility Reference for Exporting and Importing

## Switches

Use the following switches for importing and exporting model objects with Profisee.MasterDataMaestro.Utilities.exe:

| Option | Description |
| --- | --- |
| /URL:https://serverName/virtualDirectory | Specifies the Profisee server connection string to use. |
| /USERNAME:userName | Optional parameter specifying the active directory domain for the user account. Requires /USERDOMAIN and /USERPASSWORD. |
| /USERDOMAIN:domainName | Optional parameter specifying the active directory domain for the user account. Used with /USERNAME. |
| /USERPASSWORD:passwordText | Specifies the password of the user account. Used with /USERNAME. |
| /EXPORT | Exports objects from the system to a file or folder. Requires /FILE and /TYPE. Also requires /NAME to export one object at a time. When /TYPE is EXTERNALSYSTEM or TEAMS, /NAME is not used. There is only one external system list, and the only export option from the command line for TEAMS is to export all teams. When /TYPE is REPORTS, then /REPORTTYPE is also supported with possible values of SYSTEM, USER, and ALL. |
| /IMPORT | Imports objects from a file or directory in the /FILE path. Requires /FILE, which is a path to a file to import a single object, or a path to a folder to import multiple objects. |
| /TYPE:*option* | Specifies the type of object to export. Options include: - **ALL** Exports all objects. The /FILE path points to a directory. - **ADDRESS** Specifies address verification strategies as the object type. The file extension for exported address verification strategy files is \*.verification. - **APPLICATION** Specifies applications as the object type. The file extension for exported applications is \*.portalapplication. - **ARCHIVING** Specifies archiving strategies as the object type. The file extension for exported archiving strategy files is \*.archivestrategy. - **EXTERNALEVENTS** Specifies external event scenarios as the object type. The file extension for exported external event scenarios is \*.externalevent for FILE calls. - **EXTERNALSYSTEM** Specifies the external system list as the object type. The file extension for exported external system list files is \*.externalsystemlist. /NAME is not used when /TYPE is EXTERNALSYSTEM. There is only one external system list, so options for multiple objects do not apply. - **FORMS** Specifies forms as the object type. The file extension for exported forms is \*.maestroform. - **HIERARCHY** Specifies hierarchies as the object type. The file extension for exported hierarchies is `*.hierarchy`. - **MAESTROEVENTS** Specifies events as the object type. The file extension for exported events is \*.maestroevent. - **MATCHING** Specifies matching strategies as the object type. The file extension for exported matching strategy files is \*.matchingstrategy. - **MODEL** Specifies the model as the object type. The file extension for models is \*.maestromodel. *NOTE*: The CLU can only be used when there are no conflicts (metadata matches by name and GUID) and where there are no deletes. - **PRESENTATIONVIEW** Specifies presentation views as the object type. The /FILE path points to a directory. The file extension for presentation view files is \*.presentationview. - **REPORTS** Specifies Profisee Analytics reports as the object type. Supports the /REPORTTYPE switch with values of SYSTEM, USER, and ALL. For more information, see [Using the CLU to import and export reports](https://support.profisee.com/wikis/profiseeplatform/use_the_clu_to_export_and_import_reports). The file extension for exported Profisee report files is \*.maestroreport. - **RULES** Specifies data validation rules as the object type. The file extension for exported data quality rules is \*.rules. - **SUBSCRIBERCONFIGURATION** Specifies subscriber configurations as the object type. The file extension for exported subscriber configurations is \*.subscriberconfiguration. STAGINGTABLE Specifies staging tables as the object type. The file extension for exported staging tables is\*.stagingtable. - **SYNONYM** Specifies synonym lists as the object type. The file extension for exported synonym list files is \*.matchingsynonym. - **TEAMS** Specifies teams as the object type. The file extension for exported team files is \*.teams. By default, teams are imported without user memberships. Using the /INCLUDEMEMBERSHIPS switch imports teams with user memberships. - **USERS** Specifies users as the object type. The file extension for exported user files is \*.users. By default, users are imported without team memberships. Using the /INCLUDEMEMBERSHIPS switch imports users with team memberships. - **WORKFLOW** Specifies workflow as the object type. The file extension for exported workflow files is \*.workflow. |
| /FILE:"*pathToFile*" | Specifies the path of the target (when exporting) or source (when importing) file. /FILE is a single path to a file with the appropriate extension when an individual object is imported or exported. /FILE is a single directory when multiple objects are imported or exported. |
| /NAME:*objectName* | Specifies the name of the object to export. Used with /EXPORT to export a single object. |

## Examples

Use the following examples to build your own commands. Change the italicized text to match the corresponding information on your system.

| Object type | Task | CLU Command |
| --- | --- | --- |
| **All** | Import all objects of all types from a directory | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"*c:\folder*" |
| | Imports all files of all types from a directory and includes user and team assignments | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"*c:\folder*" /INCLUDEASSIGNMENTS |
| **Application** | Export an application | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:APPLICATION /NAME:applicationName /FILE:"c:\folder\applicationName.portalapplication" |
| | Export all applications | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:APPLICATION /FILE:"c:\folder" |
| | Import an application | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"c:\folder\applicationName.portalapplication" |
| **Archiving strategy** | Export an archiving strategy to a file | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:ARCHIVING /NAME:*strategyName* /FILE:"*c:\folder\archiveFile*.archivestrategy" |
| | Import an archiving strategy | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"*c:\folder\archiveStrategy*.archivestrategy" |
| **Data Quality Rules** | Export data quality rules | See [Export and Import Data Quality Rules using the CLU](https://support.profisee.com/wikis/profiseeplatform/export_and_import_data_quality_rules_using_the_clu) |
| | Import data quality rules | See [Export and Import Data Quality Rules using the CLU](https://support.profisee.com/wikis/profiseeplatform/export_and_import_data_quality_rules_using_the_clu) |
| **External event scenario** | Export an external event scenario | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:EXTERNALEVENTS /NAME:"*externalEventNames*" /FILE:"*c:\folder\subscriberConfigName*.externalevent" |
| | Export all external event scenarios | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:EXTERNALEVENTS /FILE:"*c:\folder*" |
| | Import an external event scenario | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"*c:\folder\listName*.externalevent" |
| **External systems list** | Export the external systems list | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:EXTERNALSYSTEM /FILE:"*c:\folder\listName*.externalsystem" |
| | Import an external systems list | Profisee.MasterDataMaestro.Utilities.exe /URL:https://*serverName/virtualDirectory* /IMPORT /FILE:"*c:\folder\listName*.externalsystemlist" |
| **Forms** | Export a form | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:FORMS /NAME:formName /FILE:"c:\folder\listName.maestroform" |
| | Export all forms | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:FORMS /FILE:"c:\folder" |
| | Import a form | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"c:\folder\listName.maestroform" |
| **Hierarchies** | Export a hierarchy | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:HIERARCHY /NAME:FileName /FILE:"c:\folder\FileName.hierarchy" |
| | Export all hierarchies | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:HIERARCHY /FILE:"c:\folder" |
| | Import a hierarchy | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"c:\folder\FileName.hierarchy" |
| **Profisee events** | Export a Profisee event | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:MAESTROEVENTS /FILE:"*c:\folder\listName*.maestroevent" |
| | Export all Profisee events | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:MAESTROEVENTS /FILE:"*c:\folder*" |
| | Import a Profisee event | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"*c:\folder\listName*.maestroevent" |
| **Profisee reports** | Export a Profisee report | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:REPORTS /NAME:"*maestroReportName*" /FILE:"*c:\folder\maestroReportName*.maestroreport" |
| | Export all system reports | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory/EXPORT/TYPE:REPORTS/ REPORTTYPE:SYSTEM/FILE:"c\*folder*\" |
| | Export all user reports | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:REPORTS /REPORTTYPE:USER /FILE:"*c:\folder\*" |
| | Export all reports | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:REPORTS /REPORTTYPE:ALL /FILE:"*c:\folder\*" |
| | Export a report in a specific category | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:REPORTS /NAME:"reportCategory\*maestroReportName*" /FILE:"*c:\folder\maestroReportName*.maestroreport" |
| | Import a Profisee report from file | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"*c:\folder\listName*.maestroreport" |
| | Import multiple Profisee reports from a folder | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /IMPORT /FILE:"*c:\folder\listName*" |
| **Matching strategy** | Export a matching strategy | Profisee.MasterDataMaestro.Utilities.exe /URL:https://serverName/virtualDirectory /EXPORT /TYPE:MATCHING /NAME:*strategyName* /FILE:"*c:\folder\matchingStrategy*.matchingstr |
