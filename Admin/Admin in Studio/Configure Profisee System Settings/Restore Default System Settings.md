# Restore Default System Settings

Restoring system settings lets you change one or all Profisee Server settings to the default values present when the server was installed.

## Restore Individual Setting to Default

1. Click **Settings** in the main toolbar.
2. Click **System Settings** in the menu.

The Systems Settings dialog appears.
3. Right-click the setting that you want to restore.
4. Click **Restore Value**.

The original value replaces the current value in the System Settings dialog. It will appear in bold if it differs from the current setting value on the server. If the restored and current server values are the same, there is no change in the value or appearance of the text.
5. Click **Save**.

The restored default value replaces the setting on the Profisee server.

## Restore All Settings to Default

1. Click **Settings** in the main toolbar.
2. Click **System Settings** in the menu.

The Systems Settings dialog appears.
3. Click Restore Defaults.

All of the original setting values replace the current values in the System Settings dialog. Values that differ from the stored values on the Profisee server appear in bold.
4. Click **Save**.

The restored default values replace the settings on the Profisee server.

Certain settings have no default values. These settings are blank when Profisee Server is installed:

- All settings under Service Provider Keys
- All settings under Email Server
- All settings under Proxy Addresses

These settings are unaffected by the Restore Defaults action and retain any values that you have added.

No changes are saved on the server if you click **Cancel** or exit from the dialog before clicking Save.
