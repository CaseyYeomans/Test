# Matching and Survivorship

The table below displays options and functions for Matching and Survivorship in System Settings.

| Option | Function |
| --- | --- |
| Matching minimum threshold | The minimum threshold value allowed in matching strategies. |
| Maximum match group size for web stewardship | The maximum number of members that can be displayed and managed using the web portal match group control. Groups exceeding this threshold must be managed through FastApps Studio. |
| SAIM Merge Loser Limit | The maximum number of members that can be specified as losers in a SAIM merge request. |
| Set IsRestricted | Controls the Restricted setting on new matching strategy control attributes. There are two settings: - False - off - True - on |
