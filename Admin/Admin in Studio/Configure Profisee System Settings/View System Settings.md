# View System Settings

Systems Settings includes options for viewing Profisee server settings and values.

To collapse System Settings:

1. Click **Settings** in the main toolbar.
2. Click **System Settings** in the menu.

The Systems Settings dialog appears. All settings are expanded by default.
3. Click **Collapse All**.

The System Settings dialog displays settings collapsed down to the category level.

To expand all settings, click **Expand All**.
