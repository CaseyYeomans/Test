# Remove a Link From a Control

You can remove the link from[two linked controls](https://support.profisee.com/wikis/profiseeplatform/link_controls).

To remove a link:

**--or--**

1. Begin by following the procedure in [Edit a Portal Application Page](https://support.profisee.com/wikis/profiseeplatform/edit_a_portal_application_page).
2. Do one of the following:

1. On the zone configuration page, click the control containing the link to remove.
2. In the Links panel, select the link.
3. Click **Remove Link**.

The link is removed from the control.

1. On the zone configuration page, right-click the control containing the link to remove.
2. Click **Link to** in the menu.

The controls currently linked to this control display.
3. Click the link that you want to remove.

The blue shading around the link icon in the menu will disappear, indicating that the link is removed.

3. Save the portal application page.
