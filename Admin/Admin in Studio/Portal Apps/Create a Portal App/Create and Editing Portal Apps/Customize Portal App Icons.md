# Customize FastApps Portal Application Icons

## Change Default Icon

You can change the default application icon to from a selection of alternate icons to differentiate individual apps in a group.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i8e76979fda144e27.png)

1. In Profisee FastApps Studio, open the Edit panel for the app you want to change.
2. Click anywhere in the header field. The Application Context window opens.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-ief9a695f17c459f3.png)
3. Select an image from the row. Use the arrow keys to scroll through the images.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i49dd493203dd703f.png)
4. Click the desired image to select it.
5. Click **Save**.
6. The header image changes to reflect the new selection.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i6d8dff869e06e076.png)
7. Click **Close**.
8. Open FastApps Portal to confirm that the new selection appears.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-ib78628e86ff607f2.png)

## Add New Icons

You can add your own customized icons to FastApps Portal applications. To add icons from your own library:

1. Click anywhere in the header field. The Application Context window opens.
2. Click the plus sign. A browse window opens.
3. Browse to the directory containing your icon files. Images must be in the PNG file format and no larger than 500 x 500 pixels.
4. Select the desired image.
5. Click Open.

The header image changes to reflect the new selection.

## Delete Icons

1. To delete icons from the database:
2. Use the arrow keys to scroll and locate the desired icon.
3. Click to select it.
4. Click Delete.

The icon is removed from the database.
