# Delete a Portal Application

All pages, controls, and menus included in a portal application are also deleted when the application is deleted. The deletion is permanent, so be certain this is what you want to do before performing this procedure.

To delete a portal application:

1. In the navigation panel, under Administration, click **FastApps**.
2. Select the application in the list.
3. In the toolbar, click **Delete**.
4. Confirm the deletion.
5. Complete the Data Deletion Acknowledgement form, and then click **OK**.

The portal application is removed from the list.
