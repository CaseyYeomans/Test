# Edit the Portal Application Menu

The portal application menu displays links for internal and external pages. The menu is associated with all internal pages included in the portal application.

## Edit Menu Link Information

1. On the Menu tab, click **Edit** in the toolbar.

The information in the Menu tab grid is read-only until you click **Edit**.

2. Select a menu item from the list.
3. Edit the information for the menu for this portal application in the Profisee Portal.

This information determines the way the menu items for this application display and function in the portal.

1. In the **Name** column, enter the text that should display for the portal application menu item.

The characters in the Name field allow users to recognize the application in the Portal menu.

The default page names are editable.

2. In the **Type** column, select **Internal** if the page is located under the Profisee Portal or **External** if the page is located on a different site.
3. In the **Page** column, select the page from the list to link with the menu item (for internal pages), or type the URL of the page to access (for external pages).
4. Click **Save** to save your changes.

## Delete a Menu Item

- To delete a menu item, select a menu link in the list, and then click **Delete** in the Menu tab toolbar.

## Change the Menu Order

- To change the order of the menu links, select a link in the list and then use the up and down arrows to change the item position.
