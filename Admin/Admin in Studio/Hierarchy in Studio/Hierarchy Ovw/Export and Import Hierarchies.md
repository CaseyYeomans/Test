# Export and Import Hierarchies

The Hierarchies feature area includes the option to import hierarchies and export them to other locations, such as additional servers or file sharing folders. This process provides a way to back up hierarchies and transport them between servers.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-if294bb85928902c3.png)

## Import Selected Hierarchies

To import one or more hierarchies:

1. In the navigation panel, under Administration, click **Hierarchies**.
2. In the Hierarchies toolbar, click **Import/Export**.
3. On the drop-down menu, click **Import**.
4. Browse to the file location where the hierarchies are saved.
5. Select one or more hierarchies to import.
6. Click **Open**.

The hierarchies are added to the list.

## Export Selected Hierarchies

To export one or more hierarchies:

1. In the navigation panel, under Administration, click **Hierarchies**.
2. Select one or more hierarchies in the list.
3. In the Hierarchies toolbar, click **Import/Export**.
4. On the drop-down menu, click **Export**.
5. When exporting a single file, browse to the desired folder location and select a file name to save the definition.

When exporting multiple files, browse to the desired folder location.
6. Click **OK**.

The hierarchies are saved in the selected location with the view name and the file extension \*.hierarchy.
