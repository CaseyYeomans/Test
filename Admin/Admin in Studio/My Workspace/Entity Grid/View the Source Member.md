# View the Source Member

Right-click any domain-based attribute value in any pane to open the associated entity filtered for the selected value.

To view the source member for a value:

The associated entity opens in an entity view tab, filtered for the selected value.

1. Right-click any value for a domain-based attribute (example: Match Status).
2. Click **View Source Member** on the menu.

Using View Source Member is a quick way to open the entity for a parent attribute.
