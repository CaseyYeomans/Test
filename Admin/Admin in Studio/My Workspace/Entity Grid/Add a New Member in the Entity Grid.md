# Add a New Member in the Entity Grid

To add a new member in the entity grid:

1. Open an Entity view tab.
2. Click **Add a new member** on the [entity grid toolbar](https://support.profisee.com/wikis/profiseeplatform/profisee_toolbars).

**--or--**

Right-click anywhere in the entity grid and then click **Add New Member** on the menu.

The Member window opens.

3. Type a unique code for the new member, and any other information you want to add.
4. To save your changes, click one of the following:
- **Publish** to save the changes and continue editing.
- **Publish & Close** to save the changes and close the window.
- **Publish & Add Another** to save the current member and then open a new window to add another new member.

The new member displays in the data entity grid.

When the entity grid is filtered by one or more attributes which must equal specific values, those attribute values are pre-populated in newly added members.
