# View Related Members

The View Related Members option lets you view other entities that reference the current entity. The entity opens, filtered on the member selected in the grid.

To view related members:

The entity opens in an entity view tab, filtered on the value of the member you selected in Step 2.

1. Open an entity in an entity view.
2. Right-click a member in the grid.
3. Click **View Related Members**.

A menu appears, listing one or more entities that reference the original entity.
4. Select the entity you want to open.

The entity opens in an entity view tab, filtered on the value of the member you selected in Step 2.
