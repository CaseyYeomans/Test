# Edit Link Attribute Values

To edit link attribute values:

1. In the entity grid, double-click the link attribute value you want to edit.

The Link Attribute dialog displays.

2. Type the new link address in the field.

**--or--**

Copy a link to the clipboard, and then right-click to paste into the destination.

3. Optionally, click **View** to preview the link in a browser window. Profisee automatically inserts "http" in front of link attribute values if no protocol is specified.
