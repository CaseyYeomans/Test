# Edit Members in the Member Window

1. In the entity grid, select members to edit. For more information, see [Select Multiple Members](https://support.profisee.com/wikis/profiseeplatform/selecting_multiple_members).
2. Click **Open Members** on the entity grid toolbar.
3. Make any changes to the data displayed in the window.
4. Do one of the following:

- In the Member window toolbar, click **Publish** to save the changes and continue editing the member.
- Click **Publish & Close** to save the changes and close the window.
- Click **Cancel** to abort any changes and exit.

The Member window is designed for data member comparisons and edits. Attribute names are listed in the first column, and data member information appears vertically.

Members are validated using any appropriate data quality rules whenever they are saved.

## Member Window Toolbar Options

- **Publish**--Write the member changes.
- **Copy**-- Copy selected value to the clipboard.
- **Paste**-- Paste the copied value to highlighted destination.
- **Code** {Name}-- Select the display format for domain-based attributes.

## Column Header Menu

Right-click a column header in the Member window to open the menu.

- **Auto-Size**-- Size the column width to fit the data and column label.
- **Auto-Size All**-- Size all column widths to fit the data and column labels.
- **Pin**-- When unpinned, pins selected column and all columns to the left. When pinned, unpins the selected column and all columns to the right. Refer to [Pin Columns](https://support.profisee.com/wikis/profiseeplatform/pin_members_in_the_member_window) and Reorder Columns for details on rearranging columns in the entity grid.

The popup form to edit a record is sized based on the longest value displayed. This will stretch up to 50% of your screen width before a horizontal scroll bar is displayed. Please be aware of this when viewing numbers and long set values, as it may appear as if numbers are missing.
