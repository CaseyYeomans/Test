# Viewing and Managing Workflow Activity

## Viewing Workflow Tasks

View the status of workflow tasks in the Workflow Administration tab.

### Viewing Tasks for a Workflow

1. Open the Workflow Manager tab.
2. Select a workflow from the list.
3. Click **Tasks** in the toolbar, or right-click your selection and then click **Manage Tasks** on the menu.

The tasks associated with the workflow display, including all workflow instances for all versions of the selected workflow.

### Viewing Tasks for a Workflow Instance

1. Open the Workflow Manager tab.
2. Select a workflow in the list
3. Click **Manage workflow instances for the selected workflow** in the toolbar, or right-click your selection and then click **Manage Workflows** on the menu.

The workflow instances display.
4. Select a workflow instance.
5. Click **View Tasks** in the menu bar, or right-click your selection and then click **View Tasks** on the menu.

The tasks associated with the selection display.

## Viewing Workflow Instances

To manage workflow instances:

1. Open the Workflow Administration tab.
2. Select the workflow in the list, and then click **Manage workflow instances for selected workflow** in the Workflow Activities toolbar.

You can also right-click the workflow and then click **Manage Workflows** on the menu.

The Workflows panel opens.
3. You can view the following information about currently running workflow instances:

- ID
- Name
- Version
- Status
- Initiating User
- Member Code
- Started
- Last Activity
- Open Tasks
- Completed Tasks
- Expired Tasks
- Canceled Tasks
4. To view details about current tasks, click the linked numbers for these items:

- Open Tasks
- Completed Tasks
- Expired Tasks
- Canceled Tasks

Information about the tasks for that workflow instance displays.
5. Click **Close** to close the workflow and task instances panels.

## Deleting Workflow Instances

To delete a workflow instance:

1. Open the Workflow Administration tab.
2. Select the workflow in the list, and then click **Manage workflow instances for selected workflow** in the Workflow Activities toolbar.

The Workflows panel opens.

You can also right-click the workflow and then click **Manage Workflows** on the menu.
3. Select one or more workflow instances in the list.
4. Click **Delete** on the Workflows panel toolbar.
5. Confirm the deletion.

The selected workflows are removed from the list.

Running workflow instances cannot be deleted.
