# Enable and Disable a Workflow

To disable a workflow:

1. Open the Workflow Manager tab.
2. Expand the workflow, and then right-click the workflow version.
3. Click **Disable** on the menu.
4. To re-enable the workflow, right-click the workflow and then click **Enable** on the menu.

Participants can continue to interact with running workflow instances when the workflow is disabled.

You cannot initiate a new instance of a disabled workflow.
