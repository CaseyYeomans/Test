# Profisee Address Verification Activities Overview

The **Profisee Address Verification Control** activities that are available in the Visual Studio Toolbox provide the workflow designer the tools they need perform location, mailing, name, email, and phone verification from within a workflow.

The table below lists the available activities and provides a summary of their functions and purpose. Refer to each activity in the table above for more specific usage and configuration details.

| | |
| --- | --- |
| **Activity** | **Description** |
| [ProcessAddressStrategy](https://support.profisee.com/wikis/profiseeplatform/process_address_strategy) | Supports the ability to invoke the processing of a specified **Address Verification Strategy** within a given workflow. |
| [VerifyRecordAddress](https://support.profisee.com/wikis/profiseeplatform/verify_member_address_activity) | Supports the ability to verify the address of a specific [Record](http://support.profisee.com/wikis/profiseeplatform/member_data_type) within a given workflow. |
