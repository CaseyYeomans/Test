# Restoring Default Settings

To reset your Profisee Desktop settings to the default settings:

1. Click **Settings** in the main toolbar.
2. Click **Desktop Settings** on the menu.

The **My Profisee Desktop Settings** dialog displays.
3. Click **Reset**.

When you click Reset, Profisee closes your workspace and administration tabs. Profisee restarts to apply the new settings.

All settings on all tabs of the Profisee Settings dialog reset to the default settings (not just those on the currently displayed tab).
