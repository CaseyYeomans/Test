# Configure My Profisee Desktop Settings

You can configure Profisee Desktop to suit your personal preferences.

To access the Profisee Desktop Studio settings:

1. Click **Settings** on the right side of the main toolbar.
2. Click Desktop**Settings**.

The **Desktop Settings** dialog appears.

These settings apply only to the locally installed Desktop environment and are not applied elsewhere.
