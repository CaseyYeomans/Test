# Fonts and Styles Settings

The following settings are available in the Fonts and Styles tab of Desktop Settings.

**General Fonts**:

| Option | Function |
| --- | --- |
| Hierarchy member font | Controls the font appearance for hierarchy members displayed in the hierarchy pane. |

**Grid Fonts**:

| Option | Function |
| --- | --- |
| Odd and even row fonts | Controls the font appearance for the even rows and the odd rows. You can select different fonts for even rows and odd rows to enhance readability. |
| Header row font | Controls the font appearance for the header row (column headings) in the entity grid. |
| Disabled row font | Controls the font appearance for read-only rows in the entity grid. |

**Grid Backgrounds**:

| Option | Function |
| --- | --- |
| Even and odd row backgrounds | Controls the background color for the even and odd rows. You can select different background colors for even rows and for odd rows to enhance readability. |
| Even and odd disabled row backgrounds | Controls the background color for disabled rows (as for committed versions) in the entity grid. |

- **Font selections**: Click inside the Sample Text field, or click **Edit** to open the Font dialog. Choose a font, text style, size and add any text effects, such as underline. Click **Save** to apply the new settings.
- **Row background colors**: Click inside the Sample Text field, or click **Edit** to open the Color dialog. Choose between basic colors or customize your own. Click **Save** to apply the new settings.

**Grid Borders**:

| Option | Function |
| --- | --- |
| Show row borders | Controls the appearance of borders between rows in grids. |
| Show column borders | Controls the appearance of borders between columns in grids. |
