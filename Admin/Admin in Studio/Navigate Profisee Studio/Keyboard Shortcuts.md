# Keyboard Shortcuts

| Shortcut | Function |
| --- | --- |
| CTRL + double-click the column separator in the entity grid | Auto-size all columns in the grid to fit the widest cell value. |
| CTRL + Edit a cell while other cells are highlighted | Update the same attribute value for all currently highlighted members. |
| CTRL + drag a member in hierarchy | Move the selected member as a sibling of the target member in an explicit hierarchy. |
| CTRL + Up/Down Arrow | Go to the first or last member when the grid is active. |
| CTRL+Z | In the entity grid view, undo the last action. |
| CTRL+Y | In the entity grid view, redo the last action. |
| CTRL+C | In the entity grid view, copy the current selection to the clipboard. |
| CTRL+X | In the entity grid view, cut the current selection and place it on the clipboard. |
| CTRL+V | In the entity grid view, paste the selection on the clipboard into the grid. |
| CTRL + TAB | Go to the next workspace tab on the right. |
| CTRL + SHIFT + TAB | Go to the next workspace tab on the left. |
| ALT + ENTER | Open or close the Member window (when a member is selected). |
| F2 | Enter editing mode in the entity grid (edits the selected cell). |
| F1 | Open the context-sensitive help. |
