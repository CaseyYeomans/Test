# Profisee Toolbars

The main toolbar provides access to the following functions from any tab in Profisee desktop where the function is applicable.

## Main Toolbar

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-ic701e047394e29fd.png)**Open a workspace**--Open a saved workspace or open a saved workspace into the current workspace.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-ifd072fa08f5fb344.png) **Save the workspace**--Save as a workspace .mdml file.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-ia0b638c616a82b23.png) **Close the current workspace**--Close the open workspace (after first prompting you to save any changes).

Click a link below to learn more about workspaces:

- [Workspace View Types](https://support.profisee.com/wikis/profiseeplatform/workspace_view_types)
- [Open a Saved Workspace](https://support.profisee.com/wikis/profiseeplatform/open_a_saved_workspace)
- [Open a New View Tab](https://support.profisee.com/wikis/profiseeplatform/open_a_new_view_tab)

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-ifcaa97aa450f5e4f.png) **Connect to a server**--Open the **Connect** dialog Profisee server. Use to connect to a different server, or reconnect to the current server.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i9f8a184e76aa8bd1.png) **Refresh the selected tab area**--Refresh the data displayed on the current view.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i34061b8bd7f4d1e6.png) **Settings**--Open the Settings dialog.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-iaf5b7c0152208e98.png)**Help**--Open the Help menu. Four options are available:

- ![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-iaf5b7c0152208e98.png) **Help**--Get help using Profisee.
- ![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i786b0a5a90437387.gif) **About Profisee** --View version information for Profisee.
- ![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i95eae27933bf7569.png) **View Server Status**--Open a web page on the current server to verify the current server status and configuration.
- ![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-iaf5b7c0152208e98.png) **Contact Support**--Open a webpage and log in to the Profisee website to initiate a support ticket.

## Entity Grid Toolbar

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i4f02eadd306acc14.png) **Publish**--Publish all pending changes.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i68d040b38166c7d7.png) **Undo local changes**--Undo changes in the current grid view.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i19e57760a8b1725d.png)**Copy**--Copy members or member data to the clipboard.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i8791e854bf4f3649.png)Paste--Paste clipboard data as member updates or new members.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i5b659e7edf9b51b9.png) **Open selected members**--Open the Member window for currently selected entity grid members.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-iea3d61230e6822f4.png)**Add new member**--Add a new member to the bottom of the page.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i35b7260626efac47.png)**Clone selected members**--duplicate selected members and open them in the Member window for additional changes.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i1cfea06390a4801b.png) **Delete selected members**--Remove the selected members from the entity.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i9b7dc5fad766bf9f.png)**Import**--Open the Import dialog and establish a connection to an external source for importing data.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-id66e9a1d548efb78.png)**Open the displayed data in Excel**--Use Excel to work with entity data. Three options are available:

- **Open in Excel**--Open the current page only in Excel
- **Save as Excel**--Save the current page only
- **Save entire entity**--Open the current entity in Excel

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i145349f8a59f841d.png)**Details**--Open the Member Details pane to view validation issues, transactions, or annotations.

**Code {Name}**--Toggle code and name display orientation for displayed attributes.
