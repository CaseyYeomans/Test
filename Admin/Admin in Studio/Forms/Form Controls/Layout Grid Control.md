# Layout Grid Control

The Layout Grid is a control that lets you add a customizable grid to your form.

- Add a Column to a Layout Grid control by right-clicking the Layout Grid control and then clicking Add Column on the menu.
- The Width property of columns controls the percentage of the layout grid that the column spans.
- You can add any control to a column in a Layout Grid control.

| Property | Property Option | Description |
| --- | --- | --- |
| **Display** | Label (text) | Defines the label text. |
| | Label Font Color | Defines the color of the label text. |

| **Layout** | Maximum Width (pixels) | Defines the maximum width of the control. |
| | Minimum Width (pixels) | Defines the minimum width of the control. Note: A minimum width of 0 causes the control to automatically resize. The control still displays. Do not set the control minimum width to 0 to prevent a control from displaying. |
