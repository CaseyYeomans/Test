# Label Control

The Label control lets you add a label to the form.

| **Property** | **Property Option** | **Description** |
| --- | --- | --- |
| **Display** | Label Font Color (RGB value) | |
| | Bold (True / False) | |
| | Italic (True / False) | |
| | Size (Pixel Size) | |
| | Underline (True / False) | |
| | Text | Defines the text to be displayed in the control. |
| **Layout** | Alignment (Left | Right) | Aligns the label left or right. |
| | Maximum Width (pixels) | Defines the maximum width of the control. |
| | Minimum Width (pixels) | Defines the minimum width of the control. A minimum width of 0 causes the control to automatically resize. The control still displays. Do not set the control minimum width to 0 to prevent a control from displaying. |
