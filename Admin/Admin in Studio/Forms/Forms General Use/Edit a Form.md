# Edit a Form

To edit an existing form:

1. Navigate to **Administration | Forms**.
2. Select the form in the list.
3. Click **Edit** in the Forms toolbar.

The Forms edit panel opens.
4. Make any changes to the form.
5. When you have finished making your changes, click **Save and Close**.

For information on how to configure the properties on a control tab, refer to [Use Form Controls](https://support.profisee.com/wikis/profiseeplatform/use_form_controls).
