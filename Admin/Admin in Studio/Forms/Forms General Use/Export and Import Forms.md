# Export and Import Forms

## Export Forms

To export one or more forms:

1. Navigate to **Administration | Forms**.
2. Select one or more forms in the list.
3. In the Forms toolbar, click **Import/Export**.
4. On the menu, click **Export**.
5. Browse to the file location to save the form file.

Multiple form files are saved to a single file.

1. Click **Save**.

The forms are saved in the selected location with the form name and the file extension \*.maestroform.

## Import Forms

To import one or more forms:

1. In the navigation panel, under Administration, click **Forms**.
2. In the Forms toolbar, click **Import/Export**.
3. On the menu, click **Import**.
4. Browse to the file location where the forms are saved.
5. Select a form file to import.
6. Click **Open**.
7. On the Import Form dialog, select the forms to import.

A single exported form file can contain multiple forms.

1. Click **OK**.

The form opens in the Form edit panel.

This occurs only when a single form is imported. When multiple forms are imported, you have the option to select the forms in the file to import.

1. Make any changes to the form, and then click **Save and Close**.

The form displays in the list of forms.
