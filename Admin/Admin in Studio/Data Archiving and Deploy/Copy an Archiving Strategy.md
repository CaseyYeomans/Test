# Copy an Archiving Strategy

Copying an existing archiving strategy can save you time if you need to create a new strategy that is similar to an existing one.

To copy an archiving strategy:

1. In the navigation panel, under Administration, click **Data Archiving and Deployment**.
2. On the Data Archiving and Deployment tab, click **Archive data**.
3. Select the archiving strategy.
4. On the Data Archiving toolbar, click **Copy**.

The edit panel opens.
5. Type a unique name for the strategy.

Each archiving strategy on the server must have a unique name.

6. Edit the strategy.

You can edit the following areas of an archiving strategy:

- Name
- **Filter by Last Updated On** selections
- Version
- Model objects to archive
7. Click **Save and Close**.

The copied strategy appears in the list.
