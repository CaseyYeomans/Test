# Export and Import Archiving Strategies

Exporting and importing strategies makes archiving strategies portable.

## To Export a Strategy:

1. In the navigation panel, under Administration, click **Data Archiving and Deployment**.
2. Click **Archive data**.
3. Select one or more strategies in the list.
4. On the strategy edit toolbar, click **Import/Export**.
5. On the menu, click **Export**.
6. Browse to a location to save the archiving strategy.
7. Click **OK**.

The strategies are saved in the selected location with the file extension \*.`archivestrategy`.

## To Import a Strategy:

1. Connect to the where you want to deploy the archiving strategies.
2. In the navigation panel, under Administration, click **Data Archiving and Deployment**.
3. Click **Archive data**.
4. On the strategy edit toolbar, click **Import/Export**.
5. On the menu, click **Import**.
6. Browse to the location where the strategies are saved.
7. Select one or more strategies to import.
8. Click **Open**.

The imported strategies appear in the list.

For information on importing and exporting strategies using the Command Line Utility, see [Command Line Utility Reference for Exporting and Importing](https://support.profisee.com/wikis/profiseeplatform/command_line_utility_reference_for_exporting_and_importing).
