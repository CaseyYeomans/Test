# Delete Attributes

Delete an attribute from an entity using the Advanced Modeling:

The attribute is removed from the model.

1. Expand the model to the **Entities** container.
2. Expand **Entities** in the tree.
3. Expand the entity containing the attribute to be deleted.
4. Do one of the following:

1. In the model tree, expand **Attributes**.
2. Right-click the attribute you want to delete.
3. Click **Delete** on the menu

**--or--**

4. In the model tree, select **Attributes**.
5. In the modeling grid, select the attribute you want to delete.
6. Click **Delete** on the modeling grid toolbar.

You cannot delete the Name or Code attributes from an entity.
