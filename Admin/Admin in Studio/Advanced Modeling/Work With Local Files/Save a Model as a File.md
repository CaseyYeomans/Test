# Save a Model as a File

Save a model using Advanced Modeling:

1. Click **Save** on the modeling toolbar.
2. If more than one model is currently open, select the model you want to save from the list in the **Confirm Model Selection** dialog.
3. If you have not saved the model before, enter a file name and save location.
4. Click **OK**.

Save a model with a different name, or to save a server model as a local file:

1. Open the **Save** menu on the modeling toolbar.
2. Click **Save As File** on the menu.
3. If more than one model is currently open, select the model you want to save from the list in the **Confirm Model Selection** dialog.
4. If you have not saved the model before, Profisee prompts you to enter a file name and save location.
5. Click **OK**.
