# Apply and Upgrade Licenses

## New Cluster

To apply a license to a new cluster:

1. Open Configuration Manager.
2. Select **Create a new cluster**.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i6482dbb47c214cbe.png)
3. Click **Add or Replace License**.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i414a72a90d5616a7.png)
4. Browse to the folder containing your licenses. Click the desired license to select it.
5. Click **Open**. A message window opens.
6. Click **OK**.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i3a47a3401d73021e.png)

## Existing Instance

1. Open Configuration Manager.
2. Click **Edit an Existing Instance**.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i3144fed6b0ca34f6.png)
3. Follow steps 3 - 6 above.
