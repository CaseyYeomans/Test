# Preview License Limitations

To view license information before configuring the server:

1. Open Configuration Manager.
2. Select Edit an existing instance.
3. Click Edit this instance.
4. Click **Add or Replace License**.
5. Browse to the folder containing your licenses.
6. Click the desired license to select it.
7. Click **Open**.
8. Click **OK**.
9. The view window populates with the license information. Use the scroll bar to scroll up or down.
