# License Limit Notifications

License limit notifications occur live in the workspace, and persist in the About and Help menus. The notifications display information about the specific limits of your license, and how much of your limit is currently used. See [Set License Limitation Warnings and Errors](https://support.profisee.com/wikis/profiseeplatform/set_license_limitation_warnings_and_errors) for more information

## Workspace

When the selected threshold is reached, a warning message will appear in the upper left-hand corner of the Workspace page. In this example, the threshold is set considerably lower than the recommended value of 80%. See

## Check License Status from the About Menu

1. Navigate to **Help | About the Profisee Platform**.
2. Review and confirm license information.

![](https://Profisee.magentrixcloud.com/sys/StaticAsset/Read/file-i8f3ee689db77afe2.png)

## Check Record Count From the Help Menu

Navigate to **Help | Record Count**. An information window displays the current record count.
