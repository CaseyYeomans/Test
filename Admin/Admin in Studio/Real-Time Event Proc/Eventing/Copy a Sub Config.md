# Copy a Subscriber Configuration

A single subscriber can have multiple associated configurations. Copy a subscriber configuration to create a new configuration based on an existing one.

To copy a subscriber configuration:

1. In the navigation panel, click **Real-Time Event Processing**, under Administration.
2. On the Real-Time Event Processing tab, click **Subscriber configurations**.

The Subscriber Configurations tab displays.
3. Select the subscriber configuration in the list that you want to copy.
4. In the toolbar, click **Copy**.

The configuration tab opens for a new copy of the subscriber configuration.
5. Make the necessary changes to the new subscriber configuration.
6. Click **Save and close** to exit the subscriber configuration tab.
