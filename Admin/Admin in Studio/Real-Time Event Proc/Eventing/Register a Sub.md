# Register a Subscriber

To register a subscriber:

1. In the navigation panel, click **Real-Time Event Processing**, under Administration.
2. On the Real-Time Event Processing tab, click **Subscribers**.

The Subscriber registration tab displays.
3. Select a subscriber in the **Available Subscribers** pane.

The subscribers listed in this panel are the subscribers installed on all event consuming servers in the cluster.

If no subscribers appear in the list, make sure your subscribers are installed properly. For more information, see [Installing subscribers on the event consuming server](https://support.profisee.com/wikis/profiseeplatform/install_subscribers).
4. Click the right arrow to add the subscriber to the Registered Subscribers column.
5. Click **Save** or **Save and Close** to save the registered subscriber.

After a subscriber is registered, the next step is to create a subscriber configuration.
