# Export and Import Synonym Lists

The following importing and exporting options are available on the toolbar:

- **Importing a synonym list**: Click **Import/Export** and then select **Import** on the menu. Select one or multiple synonym list files to import, and then click **Open**.
- **Exporting a synonym list**: Select one or multiple synonym lists on the tab. Click **Import/Export** and then click **Export** on the menu. Browse for the location where you want to save the a synonym lists, and then click **Save**.
- **Export all a synonym lists for a model**:
1. Click **Import/Export** and then select **Export By Model** on the menu.
2. Select the model from the **Export from model list**, and then click **OK**.
3. Browse for the location where you want to save the lists, and then click **Save**.

Imported synonym lists automatically overwrite the existing list. If you want to save the existing list before importing a new one, export it first.
