# Configure Settings on the Phone Tab

- To return information about all fields in this tab, use the Attribute Assistant o map your Phone Number source attribute to an output attribute, and to automatically configure attributes to contain the information returned in result fields.
- To return information on selected fields, manually configure the output attributes by selecting **Create New** on the menu, or by selecting an existing attribute for each field.

## Editing Result Mappings

You can map verification service result codes on the Phone tab. For more information, see[Edit Result Code Mappings](https://support.profisee.com/wikis/profiseeplatform/edit_result_code_mappings).

Phone verification is available for the US and Canada region only.
