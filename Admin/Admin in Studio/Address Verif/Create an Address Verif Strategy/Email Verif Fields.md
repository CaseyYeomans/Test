# Email Verification Fields

The following fields are available on the Email tab of an address verification strategy for source and output attributes.

| Field | Description |
| --- | --- |
| **Email** | The updated, corrected email address. If the input email address could not be verified, then it defaults to the input email address. |
| **Domain Name** | The domain name portion of the email address, excluding the Top Level Domain, and including any changes or corrections that have been made. If the final address is "jsmith@profiseecorp.com", this field contains "profiseecorp". |
| **Mailbox Name** | The mailbox or user name portion of the email address, including any changes or corrections. If the final address is "jsmith@profiseecorp.com," this field contains "jsmith." |
| **Top Level Domain Name** | The Top Level Domain name portion of the email address, including any changes or corrections that have been made by the web service. If the final address is "jsmith@profiseecorp.com", this field contains "com". |
| **Verification Results** | A comma-separated listing of Result Codes for this email address. |
| **Verification Status** | The overall verification status for this email address. |
