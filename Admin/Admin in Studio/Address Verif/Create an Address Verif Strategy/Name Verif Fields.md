# Name Verification Fields

The following fields are available on the Name tab of an address verification strategy for source and output attributes.

| Field | Description |
| --- | --- |
| Full Name | As an input, this is one or two names, stored in a single attribute. As an output, it is the full name (the first full name). |
| Full Name 2 | The full name (the second full name, if two full names were parsed). |
| Prefix | The prefix (such as Mr. or Dr.) of any name parsed. If two names were parsed, the first of the two prefixes appears here. |
| First Name | The first name of any name parsed. If two names were parsed, the first of the two first names appears here. |
| Middle Name | The middle name of any name parsed. If two names were parsed, the first of the two middle names appears here. |
| Last Name | The last name of any name parsed. If two names were parsed, the first of the two last names appears here. |
| Suffix | The suffix of any name parsed. If two names were parsed, the first of the two suffixes appears here. |
| Gender | The gender of the first name found in the FullName field. |
| Prefix2 | The second prefix, if two names were parsed. |
| First Name 2 | The second first name, if two names were parsed. |
| Middle Name 2 | The second middle name, if two names were parsed. |
| Last Name 2 | The second last name, if two names were parsed. |
| Suffix 2 | The second suffix, if two names were parsed. |
| Gender 2 | The gender of the second first name, if two names were parsed. |
| Salutation | A generated salutation for the name. |
| Parsing Results | A comma-separated listing of Result Codes for this name. |
| Parsing Status | The overall verification status for this name. |
