# Configure Settings on the Criteria Tab

Use the Criteria tab to limit the data processed by the Address Verification strategy to a smaller subset of relevant members. The criteria is configured using algorithmic query logic. Click **New** to add a new criteria line, or click **Delete** to remove selected criteria.

To filter members to process using criteria:

1. Select the **Criteria** tab in the Address Verification strategy.

If you are creating a new strategy, then you must first complete the required information on the Strategy tab.

2. Click **New** to add a new filter condition.
3. Select an attribute in the first list to define the attribute to use to filter members.
4. Select the operator for the filter criteria in the second list.
5. Enter or select a value for the filter criteria in the third list. To filter members based on blank values, leave this field empty.
