# Obtaining a Loqate Key

Address Verification offers built-in support for international address verification through Loqate. Before you can create and use strategies using Loqate, however, you must first obtain a key to unlock the service.

Profisee must have access to this Web service endpoint to perform address verification using Loqate:

- http://saas.loqate.com/rest/?p=v

Keys for verification services are obtained directly from the companies that market the services. You must contact Loqate directly to obtain a license key. Access the Loqate Web site for more information: [http://www.loqate.com](http://www.loqate.com/).
