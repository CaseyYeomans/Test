# Connect To A Profisee Server

To connect to a Profisee server:

1. Click **Connect** on the main toolbar.
2. From the **Connection** list, select the server connection.
3. Click **Connect**.

The Profisee Desktop application connects to the Profisee server.

The current connection information displays in the lower right corner of the interface. Move the mouse pointer over the connection name for more detailed information.

For more information on Profisee connections, see [Add a new Profisee connection](https://support.profisee.com/wikis/profiseeplatform/add_a_new_profisee_connection).
