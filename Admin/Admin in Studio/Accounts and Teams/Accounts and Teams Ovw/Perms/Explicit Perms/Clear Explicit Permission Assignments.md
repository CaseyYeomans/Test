# Clear Explicit Permission Assignments

Clear an [Explicit Permission](https://support.profisee.com/wikis/profiseeplatform/assign_permissions) assignment:

1. Under Administration, click **Accounts and Teams**.
2. Do one of the following:

1. Click **Users**
2. Select the user in the list.

The User edit panel opens.

**--or --**

3. Click **Teams**.
4. Select the team in the list.

The Team edit panel opens.

3. Click the **Permissions** tab.
4. Click **Edit** in the Permissions tab.

To remove an Explicit Permission, select it from the Permissions tab and click Edit. The model tree will open with the focus on the selected object.
5. Right-click the object in the model where you want to make the assignment.

To quickly remove all access to an object, clear the check box next to the model object to clear all filters and remove all access to an entity object.
6. Click **No Access** on the menu.

All permissions at the selected level and below, including filters, are removed.
7. Click **Save**.
8. Click **Save and Close**.
