# Edit the External Systems List

To edit the Profisee Server external systems list:

1. In the navigation panel, under Administration, click **External Systems List**.
2. Make any changes to the systems in the list.

You can change the system descriptions, edit the system type, and add or delete systems.

When a system is in use, it cannot be deleted from the list and the name cannot be edited.

3. When you have finished editing the list, click **Save and Close**.
