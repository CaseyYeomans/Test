# Pre-Installation

Profisee's SDK extends the functionality provided by Profisee Platform. The Profisee SDK can be installed on any developer's environment.

## System Requirements

The SDK workflow administration components--as well as forms creation, notifications, and workflow portal components--are included in the Profisee Platform. Refer to the *Enterprise Installation and Operations Guide for the Profisee Platform 2019 R1* and the Profisee help system for more information on installing and using these components.

Refer to the SDK release notes for the specific SDK you are installing for version-specific upgrade and installation assistance.

## Software Download

Profisee SDK software can be downloaded from the Profisee web site ([www.profisee.com](http://www.profisee.com/)[)](http://www.profisee.com/) using your customer login.

## Pre-Installation Checklist

Before installing the Profisee SDK, ensure that the following requirements are met or that the requested information is available. This list is a convenient summarization only; refer to other supporting sections of this document for requirement or configuration details.

- Profisee Server is installed and you have a user account to access the system having sufficient roles for application building and workflow registration and sufficient permissions to the data entities for which you want to build custom applications and workflows.
- You have downloaded the Profisee SDK installation file.
- The full .NET 4.7.2 Framework is installed on the web or application servers where any customizations implemented with the SDK may be deployed.
- You have downloaded Microsoft Web Deploy 2.0 or above (Visual Studio 2012 only)
- Workflow development computers only: If you plan to develop workflows, you have installed one of the following versions of Visual Studio: 2012, 2013, 2015 (Professional or Enterprise editions), 2017, 2019, or 2022 (Professional, Enterprise, or Community editions). Note that if you are using Visual Studio 2017, you must also install the Workflow Foundation extension.
- Workflow development computers only: You have ensured .NET 4.7.2 is installed as required to use Visual Studio toolbox and workflow templates.
