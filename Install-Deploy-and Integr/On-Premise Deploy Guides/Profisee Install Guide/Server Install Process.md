# Server Installation Process

To install Profisee Server:

1. Log in to the Web server console using a local administrator account.
2. Run the Profisee Server msi. Click **Next**.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i6b5bed8ed31575b5.png)

3. Read and accept EULA terms. Click **Next**.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-iada230daa62d6e8d.png)

4. Accept the destination location. Click **Next**

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i568a1931179a90eb.png)

While it is possible to change the drive, directory and "Profisee" subdirectory name, it is not recommended to do so.

5. Click **Install.**

6. Click **Finish**, with the **Run Profisee Server Configuration Manager ("Configuration Manager")** option selected.
